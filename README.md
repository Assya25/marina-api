# Marina API

## Description
Ce projet correspond à la réalisation d’une application web de gestion d’un port de plaisance.

L’application permet de gérer :
- les utilisateurs
- les catways
- les réservations

Elle a été développée avec Node.js, Express, MongoDB Atlas, Mongoose et EJS.

## Prérequis
Avant de lancer le projet, il est nécessaire d’avoir installé :
- Node.js
- npm

## Installation
Cloner le dépôt puis installer les dépendances avec la commande suivante :

    npm install

## Lancement du projet
Pour démarrer le serveur de développement, utiliser la commande suivante :

    npm run dev

## Technologies utilisées
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- EJS
